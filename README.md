# Movie-Review-Platform

## How It Run
 1. Clone repository:
    ```
    git clone git@github.com:DiachenkoRP/Movie-Review-Platform.git
    ```
 2. Select cloned repository:
    ```
    cd Movie-Review-Platform
    ```
 3. Up docker:
    ```
    sudo docker-compose up -d
    ```
 4. On browser go localhost:8080.
